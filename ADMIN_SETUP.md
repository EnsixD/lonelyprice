# Создание аккаунта администратора

## Шаг 1: Регистрация администратора

1. Запустите приложение локально или откройте развернутую версию
2. Перейдите на страницу регистрации: `/auth/register`
3. Зарегистрируйте аккаунт с email, который будет использоваться для администратора

**Рекомендуемые учетные данные для администратора:**
- **Email**: `admin@lonely-prize.ru`
- **Пароль**: `AdminLonelyPrize2025!` (или ваш надежный пароль)

## Шаг 2: Назначение прав администратора

После регистрации пользователя, выполните SQL скрипт в Supabase SQL Editor:

### Вариант 1: Назначить конкретного пользователя администратором

\`\`\`sql
UPDATE profiles
SET is_admin = true
WHERE id IN (
  SELECT id FROM auth.users WHERE email = 'admin@lonely-prize.ru'
);
\`\`\`

### Вариант 2: Назначить любого существующего пользователя администратором

Замените `your-email@example.com` на email зарегистрированного пользователя:

\`\`\`sql
UPDATE profiles
SET is_admin = true
WHERE id IN (
  SELECT id FROM auth.users WHERE email = 'your-email@example.com'
);
\`\`\`

## Шаг 3: Вход в систему

1. Войдите в систему используя учетные данные администратора
2. После входа вы увидите кнопку "Админ-панель" в шапке сайта
3. Перейдите в `/admin` для доступа к панели управления

## Возможности администратора

Администратор имеет доступ к:

- **Управление пользователями** (`/admin/users`)
  - Просмотр всех зарегистрированных пользователей
  - Блокировка/разблокировка пользователей
  - Удаление пользователей
  - Назначение прав администратора другим пользователям

- **Управление услугами** (`/admin/services`)
  - Создание новых услуг
  - Редактирование цен, описаний, категорий
  - Активация/деактивация услуг
  - Удаление услуг

- **Управление заказами** (`/admin/orders`)
  - Просмотр всех заказов в системе
  - Изменение статусов заказов
  - Чат с клиентами по каждому заказу
  - Статистика по заказам

## Безопасность

⚠️ **Важно:**
- Используйте надежный пароль для аккаунта администратора
- Не передавайте учетные данные администратора третьим лицам
- Регулярно меняйте пароль
- Аккуратно назначайте права администратора другим пользователям

## Проверка прав администратора

Чтобы проверить, является ли пользователь администратором:

\`\`\`sql
SELECT 
  u.email,
  p.full_name,
  p.is_admin,
  u.created_at
FROM profiles p
JOIN auth.users u ON u.id = p.id
WHERE p.is_admin = true;
